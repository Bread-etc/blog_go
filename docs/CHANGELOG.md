## [v1.0.0] - 2026-01-03

### ✨ 新增功能 Features

- **用户体系**:
  - 基于 JWT 的身份验证与会话管理
  - 实现 RSA 非对称加密登录，保障传输安全
  - 默认管理员账号自动初始化与密码修改功能
- **内容管理 (CMS)**:
  - **文章模块**: 支持 Markdown 内容存储、Slug 友好链接、SEO 优化字段及浏览量统计
  - **分类与标签**: 完整的分类 (Category) 管理与多标签 (Tag) 关联系统
- **运营工具**:
  - **站点配置**: 支持动态修改站点标题、副标题等全局配置
  - **友链系统**: 友链申请、审核与自定义排序 (Sort) 功能

### 🧪 单元测试 Test

- **单元测试**: 对核心 Service 层 (User, Post, Category, Tag, Link, Config) 实现了 **100% 核心逻辑覆盖**
- **测试环境**: 采用 `sqlite` 内存模式模拟真实数据库环境，确保测试的高效性与隔离性
